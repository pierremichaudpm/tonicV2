Let's fix these issues:

## 1. Fix the Close Button Not Working

The issue is that `toggleMobileMenu` might not be available when the button is clicked. Add this script right after your CSS styles and before the closing `</head>` tag:

```javascript
<script>
// Ensure toggleMobileMenu is available immediately
function toggleMobileMenu() {
    var menu = document.getElementById('mobileMenu');
    if (!menu) return;
    
    if (menu.classList.contains('active')) {
        menu.classList.remove('active');
        menu.style.display = 'none';
        document.body.style.overflow = '';
    } else {
        menu.classList.add('active');
        menu.style.display = 'flex';
        document.body.style.overflow = 'hidden';
    }
}

// Make it globally available
window.toggleMobileMenu = toggleMobileMenu;
</script>
```

## Complete Fix Summary:

1. Add the toggleMobileMenu function definition early in the page (in the head)
2. Update the CSS to ensure consistent hover states
3. If needed, use the inline fallback for the close button

The key issue is that `shared.js` loads after the HTML, so when the button is clicked, the function might not be available yet. By defining it early in the page, we ensure it's always available.
