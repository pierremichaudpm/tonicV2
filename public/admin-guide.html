🔌 Intégration Automatique

Site Web

Le contenu créé dans Strapi apparaît automatiquement sur votre site web sans intervention technique.

Emplois: /emplois.html et /emplois-en.html
Actualités: /communiques.html et /communiques-en.html

Système de Secours

Si Strapi est arrêté, le site utilise automatiquement les données locales existantes.
